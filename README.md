# 뮤직 플레이 리스트 관리 시스템

## 음악 관리 정보
- 음악 ID
- 제목
- 가수
- 앨범명
- 장르
- 재생시간
- 즐겨찾기 여부

## 플레이리스트 관리 기능
- 음악 추가
- 음악 삭제
- 삭제한 음악은 30일 동안 휴지통 보관 후 자동 삭제
- 전체 음악 목록 출력
- 제목으로 음악 검색
- 가수별 음악 확인
- 장르별 음악 확인
- 즐겨찾기 설정
- 즐겨찾기 음악 확인
- 플레이리스트 장르별 총 재생시간 확인
- 플레이리스트 장르별 음악 개수 확인
- 음악 정보 출력
- 음악 제목 오름차순 출력
- 장르별 음악 개수
- 휴지통 기능 + 30일 이후 자동 삭제

## 사용한 자료구조
- ArrayList: 현재 플레이리스트, 휴지통
- HashMap: 30일 이후 자동 삭제를 위한 hashmap
